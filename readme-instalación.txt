INSTALACIÓN

1. Descargar e instalar Nodejs 0.12+ https://nodejs.org/en/
2. ir al directorio del proyecto: "D:\htdocs\PER\NRNO (Exposición Virtual Antonio Nariño)"
3. Abrir consola/terminal CMD e instalar dependencias para node
    npm install
4. Correr Nodejs en la misma terminal con el siguiente comando (ubucado en la carpeta de proyecto)
    npm start
5. Esto creará un servidor web local en el puerto 80, ingresamos a Localhost desde el navegador y listo
